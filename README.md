# QuantumultX-Scripts

Personal Quantumult X helper scripts.

## Geo Location Checker

```ini
geo_location_checker=http://ip-api.com/json/?lang=zh-CN, https://raw.githubusercontent.com/Su-cyber-art/QuantumultX-Scripts/main/IP.js
```

`IP.js` formats the response from `ip-api.com` for Quantumult X node/location display.
